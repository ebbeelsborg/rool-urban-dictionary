# Rool Urban Dictionary 🍊

A visually enticing, modern, and responsive community-powered dictionary built on the [Rool](https://rool.app) platform. Define, discover, and share "rool" slang in a collaborative, AI-enhanced environment.

## ✨ Features

- **Collaborative Dictionary**: All users share a persistent space to define and search for words.
- **Floating Words Background**: A dynamic home page featuring real-time "rool" words floating in the background.
- **AI-Powered Usage Previews**: Get instant suggestions on how to use your defined words from the Rool AI.
- **Modern UI/UX**: Built with Svelte 5 and TailwindCSS v4 for a premium, high-performance experience.
- **Hype-Driven Design**: Vibrant orange aesthetics, glassmorphism, and smooth animations.

## 🚀 Getting Started

### Prerequisites
- [Rool CLI](https://www.npmjs.com/package/@rool-dev/cli) installed and authenticated.

### Installation
```bash
cd rool-urban-dictionary
npm install
```

### Development
```bash
npm run dev
```

### Publishing
To publish the app to the Rool platform:
```bash
npm run build
rool app publish rool-urban-dictionary ./dist
```

## 🛠️ Tech Stack
- **Framework**: Svelte 5
- **Platform**: Rool SDK & Svelte Wrapper
- **Styling**: TailwindCSS v4
- **Language**: TypeScript
- **Build Tool**: Vite
